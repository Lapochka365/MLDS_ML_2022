Что-то пошло не так

Пожалуйста, проверьте правильность заполненных данных. Среди возможных причин:

  1. Данные в формате JSON можно предоставлять только по одному автомобилю; для получения стоимости сразу по нескольким автомобилям, пожалуйста, загрузите файл в формате CSV
  2. Следует добавлять данные в диалоговое окно следующим образом: {"название признака": "значение признака", ...}
  3. Указаны не все признаки автомобиля, единственным опциональным признаком является цена (selling_price)
